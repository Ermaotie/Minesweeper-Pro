# Minesweeper (ClearBomb)

A modern, responsive, and "no-guess" Minesweeper game built with Vanilla JavaScript.

## Features

- **No Guessing Required**: The board generation algorithm ensures every game is solvable by logic alone.
- **Responsive Design**: Works on desktop and mobile.
- **Themes**: Clean, modern UI with clear interactions.
- **3 Difficulty Levels**: Beginner, Intermediate, and Advanced.

## Local Development

Just open `index.html` in your browser. No build step required!

## Deployment

### Deploy to Vercel

This project is ready to be deployed to Vercel.

1.  **Install Vercel CLI** (optional): `npm i -g vercel`
2.  **Run Deploy**:
    ```bash
    vercel
    ```
    Follow the prompts to deploy.

**Alternatively (Git):**

1.  Push this code to a GitHub/GitLab/Bitbucket repository.
2.  Import the project in your Vercel dashboard.
3.  Vercel will detect it as a static site.
4.  Click **Deploy**.

### Project Structure

- `index.html`: Entry point.
- `css/`: Stylesheets.
- `js/`: Application logic (bundled in `app.js`).
- `vercel.json`: Configuration for Vercel deployment.
